ol3 - Next Generation Web Mapping

OpenLayers brings geospatial data to any modern desktop or mobile web browser. ol3 is a complete rewrite, featuring WebGL and 3D. Like OpenLayers 2, it supports a huge variety of data formats and layer types. But unlike OpenLayers 2, it is built from scratch relying on latest browser technologies like HTML5, WebGL and CSS3.

Getting Started with OpenLayers 3 alpha 1

ol3 is still under heavy development, with many features missing. But the development team wants to give early adopters a sneak preview by releasing an early alpha.

  • Browse the examples, and see how to create a simple map.
  • Explore the API documentation. This is in an early stage. It shows the API already, but events are not yet documented, and introductory prose documentation needs to be added to classes and factories.
  • Start creating your own maps by simply including ol3 css and JavaScript in your web page: 
    <link rel="stylesheet" href="http://openlayers.github.io/en/master/build/ol.css" type="text/css">
<script src="http://openlayers.github.io/en/master/build/ol.js"></script>
  • Consider contributing to ol3.
